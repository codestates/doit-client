## 프로젝트 Doit

집중해서 쓴 시간과 그렇지 않은 시간의 차이는 큽니다.
DOIT은 집중력과 시간관리를 위한 작지만 강력한 툴입니다.

## 기본셋팅

1. 터미널 환경에서 git clone

```javascript
git clone https://github.com/codestates/doit-client.git
```

2. 터미널 환경에서 npm을 install

```javascript
npm i
```
